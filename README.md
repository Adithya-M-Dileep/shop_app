
# Shopping App

E-commerce app with Basic functionalities using Flutter and FireBase.

## Features

- Favorite Products
- Cart Products
- Place Order
- Seeing your order History
- Add new Products
- Sign-Up & Login Features
- Auto Logout when Session ends
